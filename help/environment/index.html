    官方规定系统要求¶

    “最低要求”是保证最新版游戏能够运行且运行流畅的最基本的要求，如不满足该要求可能会导致游戏无法安装、游戏帧数低甚至游戏崩溃。图形处理器（GPU）、中央处理器（CPU）型号未做强制要求，只需同等性能即可。

    基岩版¶

    最低要求（基本项目）1
    操作系统（OS） Windows 10 或更高版本
    系统版本号（Version） 19041.0(2004) 或更高版本
    系统架构 ARM, x64, x86
    图形处理器（GPU） Intel HD Graphics 4000 / AMD Radeon R5
    中央处理器（CPU） Intel Celeron J4105 / AMD FX-4100
    DirectX 版本 11
    运行内存（RAM） 4GB
    建议的要求（基本项目）
    操作系统（OS） Windows 10 或更高版本
    系统版本号（Version） 19041.0(2004) 或更高版本
    系统架构 ARM, x64, x86
    图形处理器（GPU） NVIDIA GeForce 940M / AMD Radeon HD 8570D
    中央处理器（CPU） Intel i7-6500U / AMD A8-6600K
    DirectX 版本 11
    运行内存（RAM） 8GB

    教育版¶

    最低要求（基本项目）2
    操作系统（OS） Windows 10 或更高版本
    系统版本号（Version） 16299.0(1709) 或更高版本
    系统架构 x64, x86
    图形处理器（GPU） Intel HD Graphics 4000 / AMD Radeon R5 / Nvidia GeForce 400 Series / AMD Radeon HD 7000 series
    中央处理器（CPU） Intel Core i3-3210 / AMD A8-7600 APU
    OpenGL 版本 4.4
    运行内存（RAM） 2GB

    如何查看相关系统信息？¶

    1. 按下Win+R，打开运行

    2. 输入msinfo32，然后点击“确定”

    3. 在弹出的窗口中找到“版本”项目，该字段就是系统版本号。如“10.0.22000 版本 22000”表示你的系统版本号为 22000

    4. 在弹出的窗口中找到“系统类型”项目，该字段就是系统的架构。如“基于x64的电脑”表示你的系统架构为 x64

    5. 在弹出的窗口中找到“已安装的物理内存（RAM）”项目，该字段就是当前设备的运行内存。如“8.0GB”表示你设备的运行内存为 8GB

    6. 在弹出窗口的侧边栏中展开“组件”，点击“显示”，找到“名称”项目，该字段就是当前设备的图形处理器（GPU）。

    关于安装包架构¶

    Mojang 提供了基于不同架构的 Minecraft for Windows 的安装包，因此并非所有安装包都能在你的设备上正常运行。

    共有以下四种指令集架构的系统和三种指令集架构的安装包，请根据设备做出选择：

    操作系统架构 x86 安装包 x64 安装包 arm 安装包
    x64
    x86
    arm
    arm64 3 4
