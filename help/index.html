    • 命令中的符号均为英文，输成中文符号会导致命令无法被程序识别！

    步骤3:安装

    按键盘上的Enter，蓝色方框里的黄色进度条开始走动（如图），直到进度条走到底，蓝色方框消失，且没有红字报错，表明安装成功。安装成功后先不要启动游戏，需要继续完成下一步操作

    4. 在窗口右侧找到“国家或地区”，选择“美国”

    点此直接打开设置页

    4. 找到“区域”选项中的“国家和地区”，选择“美国”

    Minecraft Education 仅支持使用微软工作或学校账户登录，不支持使用微软个人账户登录。如果您没有微软工作或学校账户，可点击下面的“如果您没有账户，请尝试演示课程。”（红框标志处，部分版本可能没有该选项），但是进入游戏后只能使用演示存档且不能够切换游戏模式，还会禁用其他教育版独有的功能玩法。

    下载下来的文件类型是zip而不是Appx？¶

    点击展开帮助

    你可以先尝试将扩展名zip改成Appx，然后再尝试安装，如果安装包无法被识别，请尝试更改下载环境，有以下两种情况

    使用浏览器下载时

    据一些用户反馈，使用360浏览器等国产浏览器下载文件时会把Appx文件认定为zip压缩文件，这里有如下建议:

    使用 Microsoft Edge、Google Chrome 等浏览器下载本站文件。

    使用 IDM 下载时

    请使用版本号小于或等于6.38的 IDM（如图），经过检验使用版本号大于6.38的 IDM 下载文件时会把Appx文件认定为zip压缩文件。

    如何更改 Minecraft 安装位置？¶

    点击展开帮助

    注意

    最好不要随意更改 Minecraft 的安装位置（默认安装位置：C盘），否则可能会有一些意想不到的错误

    在安装前选择位置

    1. 按下Win+I，打开“设置”

    2. 点击“系统”

    3. 点击左侧“存储”

    4. 在窗口右侧点击“更改新内容的保存位置”

    5. 找到“新的应用将保存到”，把选项C盘更改为其他盘，点击“应用”即可

    注

    在Windows11系统下，该方法似乎起不到作用

    1. 按下Win+I，打开“设置”

    2. 在“系统”页面下，点击“存储”

    3. 展开“高级存储设置”，点击“保存新内容的地方”

    4. 找到“新的应用将保存到”，把选项C盘更改为其他盘，点击“应用”即可

    在安装后移动到其他位置

    注

    在Windows10系统下，该方法仅适用于安装在除C盘以外的磁盘上的应用

    点此直接打开设置页

    1. 按下Win+I，打开“设置”

    2. 点击“应用”

    3. 找到“Minecraft”或“Minecraft Preview”并点击

    4. 点击“移动”，把选项C盘更改为其他盘，点击“移动”，等待进度条走完即可

    点此直接打开设置页

    1. 按下Win+I，打开“设置”

    2. 点击左侧“应用”

    3. 在窗口右侧点击“安装的应用”

    4. 找到“Minecraft”或“Minecraft Preview”

    5. 展开右边的“···”，点击“移动”，把选项C盘更改为其他盘，点击“移动”，等待进度条走完即可

    应用安装失败，错误消息: 由于程序包 Microsoft.ExampleProgram_9.999.999_x64__8wekyb3d8bbwe 与设备不兼容，Windows 无法安装该程序包。

该程序包要求 Windows.Universal 设备系列运行操作系统版本 10、0、19041、0 或更高版本。

该设备当前运行的操作系统版本为 10、0、xxxxx、xxx。 (0x80073cfd)

    安装 Minecraft 最新版本时，凡是系统版本号低于 19041.0(20H1) 的均会报错，如系统版本号过低，请在设置中检查系统更新，或参阅老旧版本 Win10 安装新版 Minecraft

    错误信息示例 
    部署操作失败，因为该程序包针对的是错误的处理器体系结构。

Windows 无法安装程序包 Microsoft.MinecraftUWP，因为此程序包需要体系结构 ARM，但此计算机具有体系结构 x64。(0x80073D10)

    根据错误信息就可以知道，是因为在下载时安装包架构选择错误，导致系统和安装包不兼容，进而导致无法安装。可以参阅以下项目：

    教育版相关问题¶

    登录

    Minecraft Education 仅支持微软工作或学校账户，不支持使用微软个人账户登录。由于 Minecraft Education 在中国无设立服务器，可能导致无法正常使用。在登录微软工作或学校账户后或选择了底下的“如果您没有账户，请尝试演示课程。”，跳出“出错了，尝试检查您的互联网连接”的对话框，请尝试科学上网。

    购买

    个体用户无法使用微软个人账户购买 Minecraft Education ，且中国大陆地区在正常情况下无法通过访问 Minecraft Education 官网进行购买。
