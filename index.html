    校验信息是什么？要怎么用？

    校验信息（如 MD5、SHA256）是用来验证下载的文件是否无误、是否完整的一串号码。当某个文件下载旁标注了校验信息时，可以把它记下来，下载了这个文件后用文件校验工具计算你所下载的文件的校验信息，和你之前记下的校验信息比较，就知道你下载的文件是否完整。如果两者相同，那么你所下载的文件是完整无误的。如果计算出来的和网上注明的不匹配，那么你下载的这个文件不完整或下载时出错。

    关于系统版本¶

    基岩版¶

    您的操作系统必须是 Windows 10 或更高版本，提供arm（仅正式版）、x86和x64三种架构的安装包。

    如果您要游玩最新版，系统版本号必须为19041.0(20H1)或更高版本。如果您使用的是低于该版本号的 Windows 10 系统，请在设置中检查系统更新，或尝试该方法：老旧版本 Win10 安装新版 Minecraft。了解更多系统要求

    UWP教育版¶

    您的操作系统必须是 Windows 10 或更高版本，系统版本号必须为16299.0(1709)或更高版本，提供x86和x64两种架构的安装包。了解更多系统要求

    Win32教育版¶

    可适用于 Windows 7 及以上系统版本，提供x86一种架构的安装包。

    但是对于Windows 7 用户：虽然 Minecraft Education 1.18 及更低版本支持 Windows 7，但从 2022 年 12 月开始，你将需要使用 Windows 8 或更高版本以确保 Minecraft Education 在你的 Windows 设备上正常运行。了解更多系统要求

    安装包适用情况¶

    Mojang 提供了基于不同架构的 Minecraft for Windows 的安装包，因此并非所有安装包都能在你的设备上正常运行。共有以下四种指令集架构的系统和三种指令集架构的安装包，请根据设备做出选择：

    操作系统架构 x86 安装包 x64 安装包 arm 安装包
    x64
    x86
    arm
    arm64 1 2
    如何查看系统版本号/架构？

    1. 按下Win+R，打开运行

    2. 输入msinfo32，然后点击“确定”

    3. 在弹出的窗口中找到“版本”项目，该字段就是当前系统版本号。如“10.0.22000 版本 22000”表示你的系统版本号为 22000

    4. 在弹出的窗口中找到“系统类型”项目，该字段就是当前系统的架构。如“基于x64的电脑”表示你的系统架构为 x64。
