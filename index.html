    欢迎来到 Minecraft for Windows 版本库¶

    我们是专注于为 Minecraft 正版玩家 与 Minecraft 开发者 提供 Minecraft for Windows（基岩版）全部版本的第三方游戏资源平台，你可以在这里下载到你所想要的任何版本，同时我们也是全网 Minecraft Appx 安装包源头提供商之一。

    版本库永久地址

    • www.mcappx.com

    • www1.mcappx.com

    请记住我们的网址，然后把它分享给你的好友！

    在下载前，请阅读以下内容

    本网站仅与 Minecraft for Windows (国际版) 有关。

    本网站提供微软应用商店上发布过的官方原版 Minecraft，安装包完全没有被修改。

    Minecraft 原版包含正版验证机制，如果你没有购买 Minecraft，将只能游玩体验版或不能进入游戏。

    Minecraft for Windows (过去称为Windows 10版) 属于基岩版，与Java版不同。

    请知悉我们的《行为规约和免责声明》。

    没有购买 Minecraft？请前往 minecraft.net 购买正版游戏。

    校验信息是什么？要怎么用？

    校验信息（如 MD5、SHA256）是用来验证下载的文件是否无误、是否完整的一串号码。当某个文件下载旁标注了校验信息时，可以把它记下来，下载了这个文件后用文件校验工具计算你所下载的文件的校验信息，和你之前记下的校验信息比较，就知道你下载的文件是否完整。如果两者相同，那么你所下载的文件是完整无误的。如果计算出来的和网上注明的不匹配，那么你下载的这个文件不完整或下载时出错。

    关于系统版本与架构¶

    您的操作系统必须是 Windows 10 或更高版本（教育版Win32-x86除外）。

    如果您要游玩最新版，系统版本号必须为19041.0或更高版本。如果您使用的是低于该版本号的 Windows 10 系统，请在设置中检查系统更新，否则只能游玩旧版本Win10最高适用版本。

    Mojang 提供了基于不同架构的 Minecraft for Windows 的安装包，因此并非所有安装包都能在你的设备上正常运行。共有以下四种指令集架构的系统和三种指令集架构的安装包，请根据设备做出选择：

    操作系统架构 可用安装包架构
    x64 x64、x86
    x86 x86
    arm arm
    arm64 arm、x64(仅Win11)、x86

    注：arm64模拟运行x64和x86应用程序

    如何查看系统版本号/架构？

    1. 按下Win+R，打开运行

    2. 输入msinfo32，然后点击“确定”

    3. 在弹出的窗口中找到“版本”项目，该字段就是当前系统版本号。如“10.0.22000 版本 22000”表示你的系统版本号为 22000

    4. 在弹出的窗口中找到“系统类型”项目，该字段就是当前系统的架构。如“基于x64的电脑”表示你的系统架构为 x64。
